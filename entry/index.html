運営方針

このページでは、参加窓口に先立ち、運営方針を公開します。

招待制と推薦制

既存会員の推薦を必須とし、信頼性の高いコミュニティを維持します。

ポイント制

  • 有益な投稿 → ＋ポイント
  • 招待承認 → ＋ポイント
  • 不正・悪質行為 → −∞（即時除籍）

ランク（鉱物モチーフ）

  • LV.0 : Graphite（要注意）
  • LV.1 : Silicon（入会したて）
  • LV.2 : Silver（安定した貢献）
  • LV.3 : Gold（信頼性高い）
  • LV.4 : Platinum（持続的活動）
  • LV.5 : Diamond（殿堂・守護者）

降格ルール

不正行為は即 LV.0。長期不在は徐々に降格します。

理念

公式との比較はせず、最小限の情報で、常に最新の信頼できるネットワークを目指します。
